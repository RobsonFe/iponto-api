## Description

Este projeto foi criado para demonstrar a utilização de um sistema de pontos chamado **iPonto** onde empresas se registram e criam usuários para gerenciar os pontos de seus funcionários.

## Setup de Instalação do Projeto e Dependências.

```bash
$ npm install
```

## Compilar e Rodar o Projeto Localmente

```bash
# Desenvolvimento
$ npm run start

# Modo de Observação
$ npm run start:dev

# Modo de Produção
$ npm run start:prod
```

## Rodar Testes

```bash
# Testes Unitários
$ npm run test

# Testes Ponta a Ponta
$ npm run test:e2e

# Testes de Cobertura
$ npm run test:cov
```
